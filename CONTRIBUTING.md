# Contributing to Grief Support Network

Thank you for your interest in contributing to the Grief Support Network!

## Ownership and Rights

**Important**: This project is owned exclusively by **awakeninginbelfast-svg**. All contributions will become the property of the owner upon acceptance.

## Before Contributing

Before making any contributions, please:

1. Contact the repository owner (awakeninginbelfast-svg)
2. Discuss your proposed changes
3. Wait for approval before proceeding

## How to Contribute

If approved by the owner:

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/AmazingFeature`)
3. Make your changes
4. Test your changes thoroughly
5. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
6. Push to the branch (`git push origin feature/AmazingFeature`)
7. Open a Pull Request

## Code Standards

- Write clean, readable code
- Follow the existing code style
- Add comments where necessary
- Test your changes before submitting

## Content Guidelines

Given the sensitive nature of grief support:

- Be compassionate and respectful
- Ensure all information is accurate
- Cite sources for medical/psychological information
- Use inclusive and non-judgmental language
- Avoid triggering content without proper warnings

## Pull Request Process

1. Ensure your code builds without errors
2. Update the README.md if needed
3. The owner will review your PR
4. Address any feedback from the owner
5. Once approved, the owner will merge your contribution

## Questions?

Open an issue or contact the repository owner directly.

## Code of Conduct

Please be respectful, compassionate, and professional in all interactions.
