# Grief Support Network

A compassionate web application providing support, resources, and information for individuals experiencing grief and loss.

## About

The Grief Support Network is a Next.js-based web application designed to help people navigate their grief journey. It provides access to crisis resources, support helplines, educational content about grief, and coping strategies.

## Features

- **Crisis Support**: Immediate access to crisis hotlines and emergency resources
- **Educational Content**: Information about grief stages and the grieving process
- **Resource Directory**: Curated list of support groups, helplines, and counseling services
- **Coping Strategies**: Practical advice for managing grief
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/awakeninginbelfast-svg/Grief_support_network.git
cd Grief_support_network
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Building for Production

```bash
npm run build
npm start
```
