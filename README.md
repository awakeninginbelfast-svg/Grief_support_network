# Grief Support Network

A compassionate web application providing resources and support for individuals navigating grief and loss.

## About

The Grief Support Network is a Next.js-based web application designed to provide:
- Educational resources about grief and healing
- Information about support groups and peer connections
- Practical coping strategies for managing grief
- Guidance on finding professional help

## Features

- **Resources**: Books, articles, podcasts, and organizations focused on grief support
- **Support Groups**: Information about online and in-person support communities
- **Coping Strategies**: Practical techniques for processing emotions and healing
- **Professional Help**: Guidance on finding qualified grief counselors and therapists
- **Crisis Support**: Direct links to crisis helplines and immediate support resources

## Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/awakeninginbelfast-svg/Grief_support_network.git
cd Grief_support_network
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Build for Production

```bash
npm run build
npm start
```
